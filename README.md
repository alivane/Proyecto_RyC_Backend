# Proyecto_RyC_Backend
Proyecto RyC backend con flask

### INSTALL VIRTUAL ENV

```bash
/usr/bin/python3.7 -m pip install virtualenv
virtualenv venv
cd venv/bin
source activate
```

### ADD FLASK

```bash
cd ../..
pip install flask
```

### RUN FLASK 

```bash
export FLASK_ENV=development
flask run
```
### SOURCE DEACTIVATE
```bash
source activate
```
