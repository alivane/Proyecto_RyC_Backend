# Proyecto_RyC_Backend
Proyecto RyC backend con flask

### INSTALL VIRTUAL ENV
```bash
/usr/bin/python3.7 -m pip install virtualenv
virtualenv venv --python=3.7
```

### ACTIVE ENVIROMENT
```bash
cd venv/bin
source activate
cd ../..
```

### INSTALL REQUIREMENTS
```bash
pip install -r requirements.txt
```

### RUN FLASK
```bash
export FLASK_ENV=development
flask run
```

### SOURCE DEACTIVATE
```bash
source activate
```
